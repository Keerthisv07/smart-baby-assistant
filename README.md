# Smart Baby Journey Assistant

An AI-powered assistant designed to help first-time mothers decide what to buy and when, reducing decision fatigue on large e-commerce platforms.

---

## Live Demo
https://keerthisv07.github.io/smart-baby-assistant/

---

## Problem
New mothers often feel overwhelmed by the vast number of product choices and lack clear guidance on what is essential at each stage of their baby’s growth.

---

## Solution
This assistant transforms product discovery into guided decision-making by generating:
- Personalized product recommendations  
- Contextual reasoning explaining why each product is suitable  
- Confidence scores to build trust  
- A structured “Now vs Next” baby plan  
- Multilingual output (English and Arabic)  

---

## Features
- AI-style recommendations with contextual reasoning  
- Confidence scoring for improved decision-making  
- Timeline-based baby planning (current vs next stage)  
- Bilingual support (English and Arabic)  
- Clean and modern user interface  

---

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Key Insight
The objective is not to display more products, but to reduce uncertainty and guide user decisions effectively at every stage of a baby’s development.

---

## Impact
- Reduces decision fatigue  
- Improves user confidence  
- Potential to increase conversion rates
